Menu admin
------------------

- Home
+ Data Master
	- Kabupaten / Kota
	- Standar
	- Indikator
	- Sub Indikator
	- Sekolah ---> ambil dari API Dapodik
	- Pengguna
		- TPMPS
		- Pengawas
		- LPMP (1 akun saja)

+ Data Setting
	- Pemetaan Sekolah - Pengawas
	- Pemetaan KPI - Sub Indikator

+ Data Operasional
	- Raport Sekolah (untuk menu admin, diawali dengan pilih sekolah by data table.)
	- Tabel Siklus	
	- Periode Siklus SPMI
	+ Data Siklus SPMI (untuk menu admin, diawali dengan pilih sekolah by data table)
    		- Pemetaan Mutu
    		- Rencana Pemenuhan Mutu
    		- Pelaksanaan Pemenuhan Mutu
    		- Audit Mutu
+ Laporan
	- Laporan siklus SPMI (untuk menu admin, diawali dengan pilih sekolah by data table. Dibuat dengan format persis seperti tabel petunjuk)
	- Analisis SPMI (melihat kinerja SPMI sekolah, menyusul)




Menu TPMPS (sekolah)
------------------

- Home

+ Data Operasional
	- Nilai Raport Sekolah --> Upload Excel (per sekolah untuk semua sub indikator dan indikator, format mengikuti Toni agar selaras dengan koda program)
	- Koreksi nilai raport
	+ Data Siklus SPMI (untuk menu admin, diawali dengan pilih sekolah)
    		1. Pemetaan Mutu (di dalamnya ada menu ajukan untuk menerima approval siklus 1. Siklus 2 & 3 belum bisa dilakukan sebelum approved)
    		2. Rencana Pemenuhan Mutu
    		3. Pelaksanaan Pemenuhan Mutu (di dalamnya ada menu ajukan untuk menerima approval siklus 2 & 3. Siklus 4 belum bisa dilakukan sebelum approved)
    		4. Audit Mutu
+ Laporan
	- Laporan siklus SPMI (dibuat dengan format persis seperti tabel petunjuk)
	- Analisis SPMI (menyusul)
